# OnTheGo: Product Overview

OnTheGo is a real-time mobility application that acts as a bridge between underutilized vehicular capacity and the growing demand for flexible transportation.

## Core Features

### 1. Live Drive Mode (Drivers)
The centerpiece of the platform. Drivers do not "post" rides in advance; they activate a live status that broadcasts their current route to the system.
- **Dynamic Routing:** The app recalculates matching possibilities if the driver takes a different turn.
- **Cost Offset:** Automatically suggests a contribution amount from passengers based on fuel prices and distance.

### 2. Instant Discover (Passengers)
Passengers enter their destination and see a live map of vehicles already moving toward that target.
- **ETA Precision:** Passengers see exactly when they will be picked up based on the driver's current speed and traffic.
- **One-Tap Request:** A simplified UX to request a seat without complex negotiation.

### 3. Smart Matching Algorithm
The backend AI doesn't just look at "Point A to Point B." It analyzes:
- **Path Intersections:** Where the passenger’s pickup/drop-off points intersect with the driver's existing path.
- **Time Deviation:** Ensures the driver is not delayed significantly (e.g., matching only if the detour is < 5 minutes).

### 4. Commuter Circles
Users can join or create private groups (e.g., "Tech Park Commuters," "Heritage Society"). 
- **Higher Trust:** Automatically prioritizes matches within known social or professional circles.
- **Group Discounts:** Potential for shared subscription models within organizations.

### 5. Seamless Safety Layer
- **Live Tracking:** All rides are monitored in real-time.
- **SOS Integration:** In-app emergency buttons for both driver and passenger.
- **Reputation Scores:** Gamified trust metrics based on punctuality, behavior, and frequency of use.

## User Journey

### For the Driver:
1. Open OnTheGo.
2. Set Destination (e.g., Work).
3. Toggle "Live Drive" ON.
4. Receive notification for a rider nearby moving to the same zone.
5. Accept and Pick up.
6. Drop off and receive cost-sharing credit automatically.

### For the Passenger:
1. Open OnTheGo.
2. Enter Destination.
3. View available seats on the map.
4. Select a driver based on ETA and Trust Score.
5. Join ride and pay via digital wallet.
